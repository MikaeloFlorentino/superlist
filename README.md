# 🛒 Superlist

Supermarket shopping list simple — sin servidores, sin cuentas, sin distracciones.

## Cómo usar

Abre `index.html` en tu navegador — literalmente eso es todo.

- Añade artículos con categoría
- Marca lo que ya agarraste
- Filtra por pendientes / completadas
- Todo se guarda en el navegador (localStorage)

## Stack

- HTML + CSS + JS vanilla
- Zero dependencias
- Zero build steps
- Zero backend
